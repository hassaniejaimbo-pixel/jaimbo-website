Facebook Feed

Auto-repost from your Facebook page to a "From Facebook" section on your website.

How it works

Facebook itself doesn't expose your posts as a public feed without an approved app review. We use a free third-party bridge that converts your public Facebook page into an RSS feed that the website can read every 30 seconds.

  1. Go to rss.app/new-rss-feed (free, no signup needed for trial)
  2. Paste your Facebook page URL: https://www.facebook.com/profile.php?id=61578836424894
  3. Click "Generate" — you'll get an RSS URL like https://rss.app/feeds/xxxxx.xml
  4. Copy that URL and paste it below

How it works

The Jaimbo News Bot is a Python script that runs on your server every hour. It pulls fresh articles from trusted RSS feeds, sends each one to an AI that fully rewrites it in a new voice, then publishes the result directly to jaimbo.live — just like you would from this admin panel.

  1. Download the news_bot.py file (included in your delivery package)
  2. Upload it to your server or computer
  3. Set your OPENAI_API_KEY environment variable
  4. Run bash setup_cron.sh to schedule it every hour automatically
  5. Articles appear on jaimbo.live within 30 seconds of each run
